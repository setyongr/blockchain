# Simple Blockchain Implementation in Kotlin

## Prerequisites

- Java JDK 1.8
- PostgreSQL

## Configuration
- Open `blockchain.DBBlockChain`
- Change the `dbName`, `dbUser` and `dbPassword`
- Save


## Run Command
### IntelliJ IDEA
For the best experience open this project in IntelliJ IDEA and run `Main.kt`
``
### Linux/Unix/MacOS
```
$ ./gradlew run
```
### Windows
```
$ gradlew.bat run
```
